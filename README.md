## Overview
HealthVault 2.0 is a redesigned prototype inspired by Microsoft HealthVault, which failed due to poor consumer adoption, complex UX, and lack of engagement.

This project reimagines HealthVault as a simple, user-friendly health dashboard with AI-assisted insights and strong privacy control.

## Why Microsoft HealthVault Failed
- Complicated and confusing user experience
- Manual and time-consuming data entry
- Low user engagement
- Weak doctorâ€“patient interaction
- Privacy concerns

## Our Solution (HealthVault 2.0)
HealthVault 2.0 focuses on simplicity, clarity, and engagement by providing:
- Easy profile creation
- Health report upload simulation
- AI-style health summary
- Risk level indication
- Actionable recommendations
- User-controlled data privacy

## Features Implemented
- User profile creation (name, age, conditions)
- Health report upload simulation
- AI health summary for example conditions:
  - Diabetes
  - Hypertension
- Risk level visualization
- Recommended next actions
- Privacy control toggle (doctor access)

## Tech Stack

## How to Run the Project
1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Enter user details and upload a sample report
4. View AI-generated health insights
